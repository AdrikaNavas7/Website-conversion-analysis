# Website-Conversion-Analysis

## Overview
This repository contains the full analytical workings.
The analysis covers 20 weeks of website visit and conversion data across 4 browsers and 10 markets.

## Files
- `Graph_Generation.R` — R script used for all chart generation
- `Graph_WeeklyConversion.png` — Weekly conversion rate trend (Weeks 0–19)
- `Graph_Browser_Conversion_Final.png` — Conversion rate by browser
- `Graph_Country_Conversion_Rate.png` — Conversion rate by market
- `Graph_Comparison_W6_W7_Browser.png` — Week 6 vs Week 7 browser comparison
- `Graph_Opportunity.png` — Market opportunity analysis
- `S_A_Written_Project_DataSet_Analysis.xlsx` — Raw dataset with summary and analysis tab

 ## Tools Used
- R (ggplot2, dplyr)
- Microsoft Excel for Analysis

## Key Opportunity
Market AA has 53.9M visits at only 13% conversion vs a 38.7% dataset average.
Closing that gap represents +13.8M additional conversions.
